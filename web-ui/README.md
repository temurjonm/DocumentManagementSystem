# DMS Web UI

Clean Next.js web interface for the Document Management System.

## Setup

```bash
npm install
cp .env.local.example .env.local
# Edit .env.local with your configuration
npm run dev
```

## Features

- Authentication with Cognito
- File upload with drag-and-drop
- Document list with status
- Search functionality
- Download documents
- Delete documents

## Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- React Query
- NextAuth.js
